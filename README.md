### [Japanpan麵包電商官網DEMO](https://22rubychou.github.io/Japanpan/)
### 使用技術

<ul>
  <li>Vue CLI 開發、Vuex 存購物車資料、VueRouter 做頁面切換 </li>
  <li>Axios串接資料庫</li>
  <li>Node.js</li>
  <li>BootstrapVue</li>
</ul>

### 功能說明

#### 前台功能
<ul>
  <li>商品瀏覽</li>
  <li>收藏商品</li>
  <li>將商品加入購物車</li>
  <li>註冊/登入會員</li>
</ul>

#### 會員後台功能
<ul>
  <li>瀏覽訂單資訊</li>
</ul>

#### 管理員後台功能
<ul>
  <li>瀏覽/更改訂單資訊</li>
  <li>新增/更改/編輯/刪除商品</li>
</ul>
